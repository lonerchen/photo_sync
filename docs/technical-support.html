Before You Contact Support

  • App version (mobile + storage server)
  • Platform and model (iOS/Android, desktop OS)
  • Steps to reproduce
  • Error message / screenshot

Quick Checklist

  • Phone and desktop are on the same LAN/Wi-Fi.
  • Storage server is running and reachable (default port 8765).
  • Local Network permission is granted on iOS.
  • If uploads fail, retry after reconnecting Wi-Fi and reopening both apps.

Data & Privacy

PhotoSync primarily transfers media inside your local network. See our Privacy Policy for details.
